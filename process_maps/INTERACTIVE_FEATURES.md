# Interactive Diagram Features

## Overview
The Process Maps module now includes a fully interactive diagram visualization with advanced features for better user experience.

## Interactive Features

### üñ±Ô∏è Mouse Controls

1. **Drag to Pan**
   - Click and drag anywhere on the diagram to move it around
   - Useful for large diagrams that don't fit in the viewport

2. **Zoom with Mouse Wheel**
   - Hold **Ctrl** (or **Cmd** on Mac) and scroll to zoom in/out
   - Zoom range: 50% to 300%

3. **Click Nodes**
   - Click on any node in the diagram to interact with it
   - Nodes highlight on hover for better visibility

### üéõÔ∏è Control Buttons

The diagram includes a control bar with the following buttons:

1. **Zoom In** (+)
   - Increases zoom level by 20%
   - Alternative: Ctrl + Scroll Up

2. **Zoom Out** (-)
   - Decreases zoom level by 20%
   - Alternative: Ctrl + Scroll Down

3. **Reset**
   - Resets zoom and pan to default view
   - Centers the diagram

4. **Fullscreen**
   - Opens diagram in fullscreen mode
   - Press **Esc** or click the button again to exit
   - Perfect for presentations or detailed analysis

## Usage Tips

### For Viewing Process Maps

1. **Navigate Large Diagrams**
   - Use drag to pan around the diagram
   - Use zoom controls to focus on specific areas

2. **Present to Teams**
   - Use fullscreen mode for presentations
   - Zoom in on specific process steps

3. **Analyze Processes**
   - Click nodes to see details (console logs)
   - Use zoom to examine connections

### For Editing Process Maps

1. **Edit Diagram Data**
   - Enter or modify Mermaid syntax in the text field
   - Preview updates automatically below

2. **Test Your Diagram**
   - Use the interactive preview to test your diagram
   - Verify all nodes and connections are correct

## Keyboard Shortcuts

- **Ctrl/Cmd + Scroll**: Zoom in/out
- **Esc**: Exit fullscreen
- **F11**: Toggle fullscreen (browser default)
